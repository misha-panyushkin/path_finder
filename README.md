path_finder
===========

Path finder class. Based on two points interaction.

Based on trigonometric circle.

___

Defines the following properties:


    startX      = ..;
    startY      = ..;
    shiftX      = ..;
    shiftY      = ..;

    angle       = ..;
    vector      = ..;

    startTime   = ..;
    endTime     = ..;
    speed       = ..;

    touches     = ..;

With following two methods:

	setStartPoint( X, Y, touches );
	setPoint( X, Y, touches );
